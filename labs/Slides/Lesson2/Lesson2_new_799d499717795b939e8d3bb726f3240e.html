What are Factors in R?

# Let's look at an example
data <- c("Male", "Female", "Male", "Female", "Male")
factor_data <- as.factor(data)
factor_data
[1] Male   Female Male   Female Male  
Levels: Female Male
  • Factors in R are used to handle categorical data (data that has a limited number of unique values, like gender, colors, etc.).
  • Factors are treated as nominal (unordered) or ordinal (ordered) categories.
  • This is especially useful in statistical modeling because R treats factors differently than numeric data.

The as.factor() Function

# Convert a character vector into a factor
char_vector <- c("red", "green", "blue", "green", "red")
factor_vector <- as.factor(char_vector)
factor_vector
[1] red   green blue  green red  
Levels: blue green red

  • as.factor() converts a character or numeric vector into a factor.

  • Factors have levels, which are the unique values within the factor. R treats them internally as integers but displays them as the original values.

  • You can see the levels of a factor by running:

    levels(factor_vector)
    [1] "blue"  "green" "red"
